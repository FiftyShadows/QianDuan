##题目

- 编写一个通用的事件监听函数

- 描述事件冒泡流程

- 对于一个无限下拉加载图片的页面，如何给每个图片绑定事件




###通用事件绑定

```
function bindEvent(ele,type,fn){
    ele.addEventListener(type,fn);
}
```



###关于IE低版本的兼容性

- IE低版本使用attachEvent绑定事件，和W3C标准不一样

- IE低版本使用量以非常少，很多网站都早已不支持

- 建议对IE低版本的兼容性：了解即可，无需深究
